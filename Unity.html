Unity

Welcome to the Unity section. Dive in, to explore the cutting-edge technology, innovative designs, and creative solutions I've brought to life using Unity Engine.

Keeper of the Shadows (releasing on Steam April 2025)

Keeper of Shadows is a dark, atmospheric puzzle game where players move objects and manipulate shadows to solve puzzles. The game features a unique mechanic where directional asset pushing is enhanced by visually rich shader work, and responsive controls immerse the player in the eerie environment. The combination of procedural generation and custom shaders adds depth to the dark atmosphere and gameplay.

Sleep No More: A Fun-Filled 2D Metroidvania Platformer

Sleep No More is a 2D platformer where players must navigate through shadowy environments, utilizing wall jumps and dashes to evade enemies and overcome obstacles. The game features complex mechanics such as predictive enemy movement and platforming abilities, all within hand-crafted environments. Unique visual and lighting effects enhance the mood, creating a tense and engaging experience.

Magecraft Timber Troupe

Play it on the Ludum Dare 55 site

Magecraft Timber Troupe is a fast-paced, action-packed adventure game created during a game jam. Players interact with a shop system, purchase weapons, and cast spells in a magical world brought to life through custom sprite animations. The time pressure and collaborative effort made this game a fun and creative experience.

The Gift

The Gift is a puzzle game that challenges players with logic-based puzzles involving pressure plates, object placement, and a fictional language. Players must solve intricate mini-games and logic puzzles to advance, making it a thought-provoking and immersive experience.

Better Player Mechanics

Better Mechanics focuses on providing players with precise and intuitive movement mechanics. From math-based camera controls to crouching and ladder movement, this game emphasizes smooth player interactions. The game also includes a raycast-based item pickup system and custom crosshair UI for better player feedback.

  • Learn more

Cardkyuu Game Jam

Cardkyuu is a sports game developed during a game jam, where players engage in volleyball-like gameplay. It features a state machine for various player actions such as setting, spiking, and receiving, with realistic ball physics adding to the challenge. Team collaboration and task management were key to the successful completion of the game.

ShadowVille

Shadow-Ville is a visually appealing environmental design project created during a game jam. The game takes place on a floating island, where players explore and interact with the world’s assets. The project emphasizes the integration of assets and creative use of time and resources during game jams.

Cube Brain

Cube-Brain is a 3D puzzle game where players manipulate a cube that rotates along different axes to solve puzzles. Procedural generation of tiles adds variety to each playthrough, and a dynamic camera system enhances the player’s immersion by zooming in at key moments.

  • Learn more

3D Controller

3D-Controller is a platformer that focuses on precise player control and movement mechanics. Players can perform multiple jumps and navigate complex levels designed to challenge their movement skills. The game is a foundational project that explores the core aspects of 3D game development, particularly player movement and level design.
